# GymTeam Clone — Фитнес-лендинг Кати Усмановой

> **Задание:** Взять лендинг [usmanovafit.gymteam.ru/mainpage](https://usmanovafit.gymteam.ru/mainpage#form) и повторить три экрана в вайбкодерском инструменте. Собрать чисто, аккуратно, корректно на телефоне, кнопки и формы — рабочие.

### Доработки от себя

- **Главная страница:** Добавлена логика кнопок — часть из них не работала на оригинальном лендинге (мёртвые ссылки, отсутствующие якоря). Все CTA-кнопки теперь ведут к форме записи и корректно скроллируют.
- **Экран оплаты:** Функционал, отвечающий за серверное взаимодействие и переход на другие экраны, заглушён тостом **«Станет доступно позже»** — чтобы пользователь видел обратную связь вместо молчаливого бездействия кнопок.

## Стек и инструменты

- **Инструмент сборки:** OpenCode (AI-ассистент)
- **Реализация:** Чистый HTML + CSS + JavaScript (jQuery), без фреймворков
- **Локальный сервер:** `serve` (Node.js)
- **Время работы:** ~3 часа

---

## Как запустить

```bash
npm install
```

Затем запустите скрипт для вашей ОС:

- **Windows:** `start.bat` — поднимает локальный сервер и открывает браузер
- **macOS / Linux:** `bash start.sh` — аналогичный скрипт для Unix-систем

## Что было сделано

### 1. Локализация всех ресурсов
- Шрифты (Gilroy, Roboto, Open Sans, Manrope, Glyphicons, FontAwesome 4/5) скачаны и подключены локально
- Все изображения и иконки перенесены в локальные директории (`assets_main/`, `assets_top5/`, `payment_files/`)
- Внешние CDN-зависимости (Google Fonts) заменены на локальные `@font-face`

### 2. Очистка от телеметрии
- Удалены: Яндекс.Метрика, Facebook Pixel, Google Analytics, CloudPayments SDK, reCAPTCHA
- Удалены: Gemini Agent iframe, виджет уведомлений с серверными вызовами
- Удалены: комментированные трекеры (`stat.js`), `tag.js`, `tag_phono.js`

### 3. Переподключение форм
- Форма записи на `june_top5.html` отправляет данные на `/api/submit` вместо оригинального сервера GetCourse
- Добавлен перехватчик `ajaxCall()` с конфигурируемым эндпоинтом (`window.API_CONFIG`)
- Валидация формы работает автономно, без внешних вызовов

### 4. Адаптивность
- Корректное отображение на десктопе (1440px), планшете (768px), мобильном (375px)
- Модальные окна корректно масштабируются
- Тач-кнопки имеют достаточный размер для нажатия (min 44px)

## Рабочий функционал

- Кнопки CTA плавно скроллируют к форме записи
- Таймер обратного отсчёта работает в реальном времени
- Появляются уведомления «Кто-то купил» (клиентская генерация)
- Модальное окно формы открывается/закрывается (клик, ESC, backdrop)
- Форма валидирует поля (имя, email, телефон) и отправляет POST на `/api/submit`
- Уведомление об успешной отправке отображается без перенаправления

---

## Примечание

Проект является учебным клоном лендинга для портфолио. Все серверные вызовы GetCourse перенаправлены на локальный эндпоинт. Социальные ссылки (Telegram, VK, Max) оставлены как внешние — валидные и рабочие.
